# Vue3 Components

```
Node Version: 16 LTS
Framework: Vue3
Build Tool: Vite
Library: Vuetify3、Fullcalendar、v-calendar、Moveable、Selecto、vue.draggable
```

### 在 Vue3 + Vite 的環境下，練習使用這些套件完成以下需求
- 日曆
    1. 切換月、週、日模式
    2. 自定義header(往前、往後、今天、標題、選擇模式下拉選單)
- 時間日期選擇器
    1. 日期、時間、日期 + 時間、日期範圍
- 自定義拖曳元件編輯器
    1. 元件：文字、按鈕、矩形、圖片
    2. 將列表中的物件拖曳到畫布中可以顯示相對應的元件
    3. 縮放、移動元件
    4. 元件不可超過畫布範圍、有對齊線
    5. 調整元件階層、鎖定元件
    6. 編輯元件內容：文字、顏色等
